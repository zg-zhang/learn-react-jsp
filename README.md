## JSX 简介

JSX 就是 JavaScript 和 XML 结合的一种形式。

React 发明了 JSX，可以方便的利用 HTML 语法来创建虚拟DOM，当遇到 `<` 就当作 HTML 解析，遇到 `{` 就当 JavaScript 解析。
